# Telephone Directory System

A simple **Telephone Directory System** implemented in Python using a **Singly Linked List (SSL)** data structure.

## Features

- Add new records
- Search records
- Modify existing records
- Delete records
- Display all records
- Sort records using:
  - Bubble Sort
  - Quick Sort

---

## Technologies Used

- Python 3
- Linked List Data Structure
